# QA Automation with Playwright & Qase

This repository contains automated UI tests built with Playwright.
The project is designed to integrate with Qase Test Management
and run automatically via GitHub Actions.

## Tech Stack
- Playwright
- Node.js
- Qase (Test Management)
- GitHub Actions (CI)

## Purpose
- Execute automated tests based on existing Qase test cases
- Automatically report test results back to Qase
